Writer2QML Hilfe

Einführung

Writer2Qml konvertiert Fragebögen, die mit dem Textverarbeitungsprogramm OpenOffice Writer geschrieben sind, in das Format der Questionnaire Markup Language (QML, siehe http://qml.uni-siegen.de/de). Damit eine Konvertierung möglich ist, müssen die Fragebögen unter Verwendung der QML-Formatvorlagen für Writer geschrieben werden. Die in das QML-Format übertragenen Fragebögen können genutzt weden, um mit geringem Aufwand Online Befragungen anzulegen.

Eine Anleitung zur Erstellung von Writer Dateien mit den QML-Formatvorlagen für Writer finden Sie auf der QML-Homepage (http://qml.uni-siegen.de/de).

Konvertieren von Writer Dateien

Um eine Writer Datei zu konvertieren, wählen Sie im Menü Datei * Writer Datei konvertieren oder wählen Sie alternativ das entsprechende Symbol auf der Werkzeugleiste. Wählen Sie in dem daraufhin geöffneten Dateiauswahldialog die Writerdatei aus, die Sie konvertieren möchten (Dateiendung ".odt") und bestätigen Sie Ihre Auswahl.

Falls die zu konvertierende Datei keine Fehler enthält, wird sie in direkt in das QML-Format konvertiert und gespeichert.

Die erzeugte QML-Datei wird dabei automatisch im Verzeichnis der Writerdatei gespeichert, wobei an den Dateinamen der Writerdatei ein ".xml" angehängt wird. Die QML Datei zur Writer Datei "Fragebogen.odt" wird also unter dem neuen Namen "Fragebogen.odt.xml" gespeichert.

Falls Die zu konvertierende Datei Fehler enthält, werden diese im Tab "Meldungen Writerimport" angezeigt. Wenn Sie die fehlerhaften Formatzuweisungen in der Writer Datei korrigiert und diese gespeichert haben, können Sie die Datei durch Auswahl des Symbols "Aktuelle Writerdatei wieder einlesen" erneut zur Konvertierung einlesen.

Meldungen zum Import der Fragebögen

Im Tab Meldungen Writerimport werden Statusmeldungen zum Import der Writerdatei angezeigt. Das Fenster ist zweigeteilt. Die rechte Seite zeigt eine Übersicht zu den importierten Absätzen der Writer Datei. Zu jedem Absatz wird eine fortlaufende Absatznummer, der Typ des Absatzes und der Text des Absatzes angezeigt. Im Fehlerfall wird an der entsprechenden Stelle ein Hinweis zur Art des Fehlers ausgegeben.

In der Navigationsansicht auf der Linken Seite werden die Absatznummern fehlerhaft zugewiesener Absatzformate angezeigt. Nach einem Klick auf die entsprechende Absatznummer springt die Ansicht auf der rechten Seite zur entsprechenden Stelle.

QML Ausgabe

Im Tab "Qml Ausgabe" wird die erzeugte QML-Datei in einem Textbetrachter angezeigt. Die Datei braucht nicht mehr gespeichert zu werden, dies geschieht automatisch (s.o. Konvertieren von Writer Dateien).

HTML-Ausgabe

Nach erfolgreicher Konvertierung wird im Tab "HTML Ausgabe" eine HTML Ansicht des Fragebogens angezeigt. Diese Ansicht bietet eine Vorschau auf das Aussehen des Online-Fragebogens, wie sie durch das Hochladen der QML-Datei auf den QML-Befragungsserver erzeugt wird.
